# 집 구하기 도구 PRD

## 1. 제품 개요
개인용 집 구하기 도구 - 매물 검색, AI 분석 리포트, 내 조건 기반 추천, 관련 뉴스까지 한곳에서

## 2. 타겟 사용자
- 나 (1인 사용자) - 결혼 후 집을 구하기 위한 개인 도구

## 3. 핵심 기능
- 매물명/주소 텍스트 검색 + 필터 (유형, 가격, 면적, 층수, 방수, 건축년도, 지역)
- 검색 결과 리스트 + 네이버 부동산 링크 연결
- 내 조건 설정 → 조건 매칭 매물 추천 + "최신 추천 매물 검색" 버튼
- 5개 AI 분석 리포트 (시세/입지/투자/규제/리스크)
- 리포트 이력 관리
- 매물별 상태 추적 (관심/방문예정/방문완료/후보/탈락) + 개인 메모
- 관심 매물 관련 뉴스 매일 자동 크롤링
- 매물 비교 (나란히)

## 4. 페이지 구조
/                    → 대시보드 (내 조건 요약 + 추천 매물 + 최근 리포트)
/search              → 매물 검색 (텍스트 + 필터 → 리스트)
/property/:id        → 매물 상세 + 네이버 링크 + 관련 뉴스 + 리포트 요청
/reports             → 리포트 이력
/reports/:id         → 리포트 상세
/compare             → 매물 비교
/settings            → 내 조건 설정

## 5. 기술 스택
- Next.js 16 (App Router)
- Supabase (PostgreSQL + Edge Functions + pg_cron)
- TanStack Query + Zustand
- Tailwind CSS v4 + Recharts + Framer Motion

## 6. 삭제된 기능
- 카카오맵 지도 검색
- 회원가입/로그인 (개인용이므로 불필요)
- OAuth, 프로필, RLS
