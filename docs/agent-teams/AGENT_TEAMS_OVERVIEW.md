## 개요

### 목적
부동산 매물에 대한 종합적이고 깊이 있는 분석 리포트를 생성하기 위해 5개의 전문 분석 에이전트가 협업하는 시스템입니다.

### 핵심 가치
- **다각도 분석**: 시세, 입지, 투자, 규제, 리스크 5개 관점에서 독립적 분석
- **병렬 처리**: 순차 처리 대비 4-5배 빠른 리포트 생성 (40분 → 10분)
- **교차 검증**: 에이전트 간 데이터 공유 및 상호 검증으로 신뢰도 향상
- **의사결정 지원**: 부동산 구매라는 중요한 결정을 데이터 기반으로 지원

### 적용 시나리오
1. **매물 상세 분석**: 개별 매물에 대한 종합 리포트 생성
2. **매물 비교 분석**: 2-3개 후보 매물의 항목별 비교 분석
3. **투자 시나리오 분석**: 특정 매물의 다양한 투자 전략 시뮬레이션

## 팀 구성

### 팀 리더 (Team Lead)
- **역할**: Claude Code 메인 세션
- **책임**:
  - 팀원 생성 및 작업 할당
  - 팀원 간 커뮤니케이션 조율
  - 최종 리포트 통합 및 품질 검증
  - 종합 점수 산출 및 구매 추천 의견 제시

### 분석팀 (Analysis Team) - 5명

| 팀원 | 전문 분야 | 주요 책임 |
|------|-----------|----------|
| **시세 분석가** | 시장 가격 분석 | 실거래가 비교, 호가 적정성, 시세 동향 |
| **입지 분석가** | 지리적 위치 평가 | 교통, 교육, 편의시설, 환경 점수화 |
| **투자 분석가** | 수익성 분석 | 미래 가치, 수익률, 보유 비용 계산 |
| **규제 분석가** | 법률/세금 검토 | 대출 규제, 세금 계산, 정책 영향 |
| **리스크 분석가** | 위험 요소 평가 | 건물 상태, 재건축, 혐오시설, 자연재해 |

### 2. 단일 매물 분석 프롬프트

```
부동산 매물 상세 분석 리포트를 작성할 에이전트 팀을 생성해주세요.

매물 정보:
- 주소: 경기도 수원시 영통구 광교산로 123
- 단지명: 수지 광교산 아이파크
- 면적: 84㎡ (전용면적 59㎡)
- 호가: 12억 5천만원
- 준공년도: 2018년
- 층수: 15층 / 총 25층
- 방향: 남향
- 매물 타입: 아파트 매매

팀 구성:
1. 시세 분석가 (Market Analyst)
   - 국토교통부 실거래가 API 활용
   - 해당 단지 최근 6개월 거래 동향 분석
   - 주변 3km 내 유사 매물 시세 비교
   - 호가 대비 실거래가 갭 분석
   - 시세 적정성 점수 (0-100점)

2. 입지 분석가 (Location Analyst)
   - 지하철역 도보/차량 거리 계산
   - 초중고 학군 정보 조사 (학교알리미 API)
   - 대형마트, 병원 접근성 평가
   - 공원, 문화시설 점수화
   - 소음/악취 리스크 (도로, 공장 근접도)
   - 입지 종합 점수 (0-100점)

3. 투자 분석가 (Investment Analyst)
   - 향후 개발 계획 조사 (도시계획정보 시스템)
   - 역세권/재개발 여부 확인
   - 전세가율 분석 및 갭투자 가능성
   - 5년 후 예상 시세 (과거 데이터 기반 추정)
   - 월 보유 비용 계산 (관리비, 재산세, 이자 등)
   - 투자 가치 점수 (0-100점)

4. 규제 분석가 (Regulation Analyst)
   - 조정대상지역/투기과열지구 여부
   - 대출 규제 확인 (DSR, LTV, DTI)
   - 양도소득세 시뮬레이션 (5년 보유 시)
   - 취득세, 재산세 정확한 계산
   - 청약 당첨자 규제 페널티 여부
   - 규제 리스크 점수 (0-100점, 낮을수록 규제 많음)

5. 리스크 분석가 (Risk Analyst)
   - 건물 노후도 평가 (준공년도 기준)
   - 재건축 예상 시기 계산
   - 하자 이력 조회 (가능한 범위)
   - 주변 혐오시설 확인 (쓰레기처리장, 화장장 등)
   - 침수/산사태 위험 지역 여부 (국토정보플랫폼)
   - 리스크 안전도 점수 (0-100점)

작업 방식:
- 각 팀원은 독립적으로 조사 및 분석 수행
- 중요 발견 사항은 다른 팀원에게 메시지 전달
- 모든 분석 완료 후 팀 리더가 최종 리포트 통합
- 최종 리포트에는 각 영역별 점수와 종합 점수 포함
- 구매 추천도 (강력 추천 / 추천 / 보통 / 비추천 / 강력 비추천)
- 주요 강점 3가지, 주의사항 3가지 명시

출력 형식: Markdown
```

### 3. 매물 비교 분석 프롬프트

```
- 면적: 84㎡
- 준공: 2015년

작업 방식:

- 항목별 점수 비교표 작성

- 최종 추천 순위 및 근거 제시

출력 형식: Markdown (비교표 포함)
```

## 작업 흐름

### Phase 1: 초기화 (0-1분)
1. 팀 리더가 매물 정보 파싱
2. 5개 팀원 세션 생성
3. 각 팀원에게 역할 및 매물 정보 전달
4. 공유 태스크 리스트 생성

### Phase 2: 병렬 분석 (5-8분)
```
[시세 분석가] → 국토부 API 호출 → 실거래가 수집 → 시세 비교 분석
[입지 분석가] → 카카오맵 API 호출 → 거리 계산 → 편의시설 점수화
[투자 분석가] → 개발계획 조사 → 수익률 계산 → 미래 시세 예측
[규제 분석가] → 규제 정보 확인 → 세금 계산 → 대출 한도 산출
[리스크 분석가] → 위험 요소 조사 → 체크리스트 검증 → 안전도 평가
```

### Phase 3: 교차 검증 (1-2분)
- 시세 분석가: "입지 분석가님, 역세권 여부가 시세에 20% 프리미엄 반영됩니다"
- 투자 분석가: "규제 분석가님, 조정지역이면 대출 한도가 수익률에 영향 줍니다"
- 리스크 분석가: "시세 분석가님, 재건축 예정이면 현재 시세는 의미 없습니다"

### Phase 4: 통합 리포트 작성 (2-3분)
1. 팀 리더가 5개 분석 결과 수집
2. 항목별 점수 정규화 (0-100점)
3. 가중 평균으로 종합 점수 계산
   - 시세 25%, 입지 25%, 투자 20%, 규제 15%, 리스크 15%
4. 구매 추천도 결정
5. Markdown 리포트 생성

### Phase 5: 정리 (0-1분)
- 팀원 세션 종료
- 리포트 파일 저장
- 토큰 사용량 로깅

**총 소요 시간: 8-15분**

## 주의사항

### 데이터 신뢰성
- 공공 API 데이터는 실시간이 아닐 수 있음
- 중요 정보는 반드시 공식 사이트에서 재확인 필요
- AI 분석은 참고용이며 최종 결정은 전문가 상담 권장

### 에이전트 간 통신
- 팀원 간 메시지 전달 시 중요 정보만 공유
- 불필요한 통신은 토큰 낭비 유발
- 팀 리더가 적극적으로 조율

### 비용 관리
- 한 달 10개 매물 분석 시 약 ₩35,000
- 예산 초과 방지를 위해 진지하게 고려하는 매물만 분석
- 간단한 확인은 단일 세션으로 처리
