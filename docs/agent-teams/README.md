| 사용 패턴 | 리포트 수            | 월 비용 | 비고             |
| --------- | -------------------- | ------- | ---------------- |
| 라이트    | 5개 매물             | ₩17,500 | 관심 매물만 분석 |
| 스탠다드  | 10개 매물            | ₩35,000 | 적극 탐색        |
| 헤비      | 20개 매물 + 비교 3회 | ₩92,500 | 집중 탐색 기간   |

**참고**: 부동산 중개 수수료 (매매가의 0.4-0.9%)와 비교하면 매우 저렴

**예시**:

- 12억 매물 중개 수수료: 480만원~1,080만원
- Agent Teams 분석 비용: 3,500원 (0.03%)
- **비용 대비 효과**: 1,300배 이상

## ⚠️ 중요 주의사항

### 1. 데이터 신뢰성

- AI 분석은 **참고용**이며 최종 결정은 전문가 상담 권장
- 공공 API 데이터는 1-2개월 지연될 수 있음
- 중요 정보는 반드시 공식 사이트에서 재확인

### 2. 현장 확인 필수

- 서류상 문제 없어도 **직접 방문 필수**
- 낮/밤, 평일/주말 다른 시간대 확인
- 주민 인터뷰로 체감 리스크 파악

### 3. 법률/세무 자문

- 복잡한 세금 계산은 **세무사 상담** 권장
- 대출 심사는 **은행 상담** 필수
- 계약서는 **법무사 검토** 권장
